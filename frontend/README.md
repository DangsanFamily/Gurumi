## 2. yarn 설치

### Window
```
npm install -g yarn
```
### Mac
```
brew install yarn --without-node    # node 가 설치되어있다면
brew install yarn                   # node 가 없다면
```
or
```
npm install --global yarn
```
## 3. Dependency 설치

root directory에서
```
yarn
```

## 4. Dev Server Start
```
yarn start
```
